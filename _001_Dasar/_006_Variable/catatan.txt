Variable:
* Variable adalah tempat untuk menyimpan data
* Variable digunakan agar kita bisa mengakses data yang sama berkali-kali dan dimanapun kita mau
* Di Go-Lang, variable hanya bisa mengakses tipe data yang sama, jika kita ingin tipe data yang berbeda maka kita harus membuat variable baru
* Untuk membuat variable kita bisa menggunakan kata kunci 'var', lalu diikuti dengan nama variable dan tipe datanya (Contoh: var myVar string)


Tipe Data Variable:
* Saat kita membuat variable, kita wajib menyebutkan tipe data untuk variable tersebut
* Tetapi jika kita langsung menginisialisasikan datanya, maka kita tidak perlu menambahkan tipe data untuk variablenya


Kata Kunci Var:
* Di Go-Lang, kata kunci 'var' saat membuat variable sebenarnya tidak lah wajib
* Agar tidak perlu menggunakan kata kunci 'var', kita perlu menggunakan gabungan tanda ':=' saat membuat variable
* Kita hanya perlu menggunakan saat pertama kali membuat variable, jika kita ingin merubah datanya, maka tidak perlu menggunakan tanda ':='
* Dan kita tidak boleh menyebutkan tipe datanya, akan secara otomatis dideteksi oleh Go-Lang


Deklarasi Multiple Variable:
* Di Go-Lang kita bisa membuat banyak variable secara bersamaan
* Dan kode yang kita buat akan lebih efisien dan mudah dibaca
