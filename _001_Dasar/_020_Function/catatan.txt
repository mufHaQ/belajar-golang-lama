Function:
* Kita pasti sudah tahu sebuah function yang wajib kita buat agar program kita bisa berjalan, yaitu function main
* Function adalah sebuah blok kode yang sengaja dibuat agar bisa digunakan berulang kali
* Cara membuat function sangat sederhana, hanya dengan menggunakan kata kunci 'func' lalu diikuti dengan nama function dan blok kode isi functionnya
* Setelah membuat function, kita bisa mengeksekusi function tersebut dengan memanggilnya menggunakan nama_func diikuti tanda '()', contoh: sum()
