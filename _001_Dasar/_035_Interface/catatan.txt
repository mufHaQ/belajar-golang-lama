Interface:
* Adalah tipe data Abstract, dia tidak memiliki implementasi langsung
* Sebuah interface berisikan definisi-definisi method
* Biasanya interface digunakan sebagai kontrak


Implementasi Interface:
* Setipa tipe data yang sesuai dengan kontrak interface, secara otomatis dianggap sebagai interface tersebut
* Sehingga kita tidak perlu mengimplementasikan interface secara manual
* Hal ini agak berbeda dengan bahasa pemrogramman lain yang ketika membuat interface, kita harus menyebutkan secara eksplisit akan menggunakan interface mana
